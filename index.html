⚙️ Uso general

¿Necesito pagar para usar TarotMe?

La biblioteca de las 78 cartas es completamente gratuita y no requiere ninguna cuenta ni clave. Puedes explorar todas las cartas, sus significados y simbolismo sin límite.

La lectura personalizada con IA requiere tu propia clave de API de Anthropic. Esta clave se usa directamente desde tu navegador y se guarda solo en tu dispositivo.

¿Qué modelo de IA usa TarotMe?

TarotMe utiliza Claude claude-opus-4-7 de Anthropic, uno de los modelos más avanzados disponibles. Está configurado con un prompt especializado para actuar como tarotista experimentada, interpretando las cartas en español con un estilo cálido y cercano.

¿Es segura mi clave de API?

Tu clave de API se guarda únicamente en tu dispositivo (localStorage del navegador) y nunca se envía a ningún servidor de TarotMe. Las solicitudes van directamente desde tu navegador a la API de Anthropic.

⚠️ Nunca compartas tu clave con nadie ni la introduzcas en páginas desconocidas.

🔑 Clave de API y límites de uso

¿Cuántas lecturas puedo hacer gratis?

Anthropic ofrece aproximadamente $5 de crédito inicial para cuentas nuevas. No hay una cuota fija diaria de consultas gratuitas: el crédito se va consumiendo por uso.

ConceptoDetalle
Crédito inicial (cuenta nueva)~$5 USD (uso único)
Coste por lectura (aprox.)$0.05 – $0.12 USD
Lecturas estimadas con $540 – 100 lecturas
Cuota diaria gratuita permanente❌ No existe

💡 El coste varía según la longitud de la pregunta y la respuesta generada.

¿Qué pasa cuando se acaba el crédito gratuito?

Una vez consumidos los $5 de crédito inicial, recibirás un error de tipo insufficient_quota o similar. Para seguir usando la lectura con IA necesitarás añadir un método de pago en console.anthropic.com/billing.

La tarifa de Claude claude-opus-4-7 es de pago por uso:

Tipo de tokenPrecio por millónPor lectura típica
Entrada (imagen + pregunta)$15 / M tokens~$0.01
Salida (interpretación)$75 / M tokens~$0.07
Total por lectura~$0.08 USD

Con $10 de recarga puedes hacer aproximadamente 125 lecturas. Los precios son orientativos y pueden cambiar.

¿Existen límites de velocidad (rate limits)?

Sí. Anthropic aplica límites de tasa según el nivel de tu cuenta. Para cuentas nuevas (sin historial de pago), los límites suelen ser:

  • Tier 1 (sin facturación): ~5 peticiones por minuto, 10.000 tokens por minuto
  • Tier 2 (con método de pago): límites más altos según uso

Para el uso normal de TarotMe (una lectura cada pocos minutos) estos límites no suponen ningún problema.

¿Cómo obtengo una clave de API de Anthropic?
  1. Ve a console.anthropic.com y crea una cuenta gratuita.
  2. Accede a Settings → API Keys y pulsa Create Key.
  3. Copia la clave (empieza por sk-ant-). Solo se muestra una vez.
  4. En TarotMe, pulsa el icono ⚙️ en la cabecera y pega tu clave.

🎁 Las cuentas nuevas reciben ~$5 de crédito automáticamente, sin necesidad de introducir datos de pago.

🔧 Errores comunes

Error: invalid_api_key

La clave introducida no es válida. Comprueba que empieza por sk-ant- y que la copiaste completa sin espacios extra. Genera una nueva en console.anthropic.com/settings/keys.

Error: insufficient_quota / credit balance is too low

Has agotado tu crédito. Añade un método de pago en console.anthropic.com/billing o espera si tienes créditos mensuales renovables en tu plan.

Error: overloaded_error o la lectura no responde

La API de Anthropic está temporalmente sobrecargada. Es un problema del lado del servidor, no de tu clave. Espera unos segundos y vuelve a intentarlo.
